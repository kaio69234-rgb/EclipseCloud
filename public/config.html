O que é o eclipsecloud.config?

Arquivo de configuração usado para automatizar o deploy de aplicações na EclipseCloud. Ele define informações como memória, arquivo principal e comandos de inicialização.

Parâmetros do Arquivo

MAIN

Arquivo principal da aplicação

TYPE

bot ou site

RAM

Memória da aplicação

NAME

Nome exibido no painel

APT

Pacotes extras do sistema

START

Comando de inicialização
