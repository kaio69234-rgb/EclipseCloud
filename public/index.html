Gerador inteligente para configurar aplicações

Configuração

Preview do Arquivo

MAIN=index.js

Parâmetros do Arquivo

MAIN

Arquivo principal da aplicação

TYPE

Tipo da aplicação (bot ou site)

RAM

Memória usada pela aplicação

NAME

Nome exibido no painel

APT

Pacotes extras do sistema

START

Comando de inicialização

Exemplos

Bot

NAME=Meu Bot
TYPE=bot
MAIN=index.js
RAM=100

Site

TYPE=site
MAIN=index.ts
RAM=512
START=npm run start
