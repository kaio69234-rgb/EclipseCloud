O que é o eclipsecloud.config?

Arquivo usado para configurar aplicações automaticamente na EclipseCloud. Define memória, tipo da aplicação e comandos de inicialização.

Parâmetros do Arquivo

MAIN

Arquivo principal

TYPE

bot ou site

RAM

Memória usada

NAME

Nome exibido

APT

Pacotes extras

START

Comando de inicialização
