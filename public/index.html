O que é o eclipse.config?





O eclipse.config é um arquivo de configuração que automatiza o
processo de upload e configuração de aplicações na EclipseCloud.
Com ele você não precisa preencher formulários manualmente a cada deploy.







⚠️ Importante




O arquivo deve estar na raiz do seu projeto e ser incluído no .zip enviado
para a EclipseCloud. Sem ele o upload da aplicação não funcionará.


















Parâmetros do Arquivo









MAIN

Arquivo principal






TYPE

bot ou site






RAM

Memória usada






NAME

Nome exibido






APT

Pacotes extras






START

Comando de inicialização


















Exemplos Práticos










🤖 Configuração para Bot



NAME=Meu Bot Discord
TYPE=bot
MAIN=index.js
RAM=100





Ideal para: Bots Discord, Telegram, WhatsApp etc.











🌐 Configuração para Site



ID=meusite.eclipse.app
TYPE=site
MAIN=index.ts
RAM=512
START=npm run start





Ideal para: APIs, sites web e dashboards.
