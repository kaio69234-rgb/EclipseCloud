O que é o eclipsecloud.config?


Arquivo usado para configurar aplicações automaticamente na EclipseCloud.
Define memória, tipo da aplicação e comandos de inicialização.



Parâmetros do Arquivo



MAIN

Arquivo principal



TYPE

bot ou site



RAM

Memória usada



NAME

Nome exibido



APT

Pacotes extras



START

Comando de inicialização



Exemplos Práticos

🤖 Configuração para Bot
NAME=Meu Bot Discord
TYPE=bot
MAIN=index.js
RAM=100

Ideal para: Bots Discord, Telegram, WhatsApp, etc.
🌐 Configuração para Site
ID=meusite.eclipse.app
TYPE=site
MAIN=index.ts
RAM=512
START=npm run start

Ideal para: APIs, sites web, dashboards, etc.
